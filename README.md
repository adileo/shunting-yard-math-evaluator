Shunting Yard Algorithm implementation and postfix notation expressions evaluation.

## How To:

```bash
npm install
```

Run test file:
```bash
node app.js
```

## Documentation:
[Here: docs.md](docs.md)

## Generating Documentation:
```bash
npm install -g jsdoc-to-markdown
```
And then:
```bash
jsdoc2md "*.js" > docs.md
```
